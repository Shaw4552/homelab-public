# Architecture Summary

This repository documents a multi-site, segmented network designed using enterprise-style principles.

## Key Concepts

- VLAN-based segmentation aligned to trust boundaries
- deny-by-default firewall policy
- centralized service hosting
- role-based VPN access
- consistent multi-site design

## Design Approach

The network is designed to:

- minimize lateral movement
- isolate low-trust devices
- centralize critical services
- enforce access through policy, not trust

## Outcome

The result is a structured, scalable network architecture that mirrors real-world enterprise design patterns.
