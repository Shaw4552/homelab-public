# Enterprise-Style Homelab Network Overview

This project demonstrates enterprise-style network segmentation and documentation practices in a home lab environment.

## Goals

- separate endpoints by trust level and function
- enforce firewall boundaries between user, service, and infrastructure networks
- implement secure remote access and inter-site connectivity
- centralize infrastructure services
- document the environment like a production network

## Environment Summary

The lab includes:

- a primary infrastructure site
- a secondary access site
- segmented VLANs
- wireless networks mapped to trust zones
- centralized DNS filtering and recursive resolution
- remote-access and site-to-site VPN design

## Core Skills Demonstrated

- VLAN architecture
- IP addressing design
- firewall policy design
- wireless segmentation
- VPN architecture
- DNS service placement
- infrastructure documentation
